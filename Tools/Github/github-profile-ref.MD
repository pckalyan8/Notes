## Custom Badge Template

To create a custom badge, use this format:
```markdown
![Badge Name](https://img.shields.io/badge/YOUR_TEXT-YOUR_COLOR?style=flat-square&logo=LOGO_NAME&logoColor=white)
```

**Find logos at:** https://simpleicons.org/

**Color format:** Hex code without `#` (e.g., `007ACC` for blue)

---

## Quick Update Checklist

When adding new skills to your README:

1. ✅ Find the appropriate badge from this reference
2. ✅ Update both the text section and badge section
3. ✅ Replace `yourusername` with your actual username
4. ✅ Test that badge logos appear correctly
5. ✅ Keep consistent styling (flat-square)
6. ✅ Organize by category (Frontend, Backend, Tools, etc.)

---

## Tips

- Keep badges in alphabetical or relevance order
- Use `flat-square` style for consistency
- Limit to technologies you actively use
- Update stats widget URLs with your username
- For LeetCode, verify your username is public
